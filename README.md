## Installation
#### Terminal
```
# Install dependencies at root directory:
yarn  
```
#### Configuration

Create **.env.local** at root directory:
```
NEXT_PUBLIC_KEY_DEVELOPMENT=true
```

## Usage

- At the root of the application
```
yarn dev
```
